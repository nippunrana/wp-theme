A lightweight, blazing-fast WordPress block theme by EgniTech with native dark/light mode support.

== Description ==

EgniTech One is a modern, performance-first WordPress block theme designed for speed and simplicity. Built entirely with Full Site Editing (FSE) capabilities, it requires no page builders and keeps your site ultra-fast.

**Key Features:**

* Native dark/light mode — respects OS preference, toggleable by users
* Self-hosted Inter variable font for modern, crisp typography
* Under 50 KB theme weight (excluding font)
* Zero jQuery dependency
* Full Site Editing support — customize everything visually
* Clean, responsive design for all devices
* Accessible with proper ARIA labels and focus styles
* Fluid typography and spacing with CSS clamp()
* Google reCAPTCHA v2 support — easily manage keys site-wide in Theme Options
